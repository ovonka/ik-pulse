# iK Pulse

A merchant transaction and settlement health dashboard inspired by fintech operational workflows.

## Overview

iK Pulse is a full-stack project designed to demonstrate:

- React + TypeScript frontend architecture
- Node.js + TypeScript backend services
- PostgreSQL relational data modeling
- JWT authentication
- observability concepts such as logging and metrics
- webhook and idempotency handling
- CI/CD and cloud-ready deployment practices
